# grepme-x website (branch: `grepme-x-website`)

Landing page for **grepme-x** built with React + Vite + Tailwind.

Live site: https://grepme-x.vercel.app/

## Stack

- React 19
- Vite 8
- Tailwind CSS 3
- Radix UI Tabs

## Local development

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite (usually `http://localhost:5173`).

## Available scripts

- `npm run dev` – start dev server
- `npm run build` – production build
- `npm run preview` – preview production build locally
- `npm run lint` – run ESLint

## Project structure

```text
src/
  App.jsx
  components/
    announcement-banner.jsx
    ui/
  lib/
    utils.js
```
